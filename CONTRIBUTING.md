# Contributing to AI Commerce Agent

Thank you for your interest in contributing to AI Commerce Agent.

This project is in its early stage and focuses on building practical AI-powered workflows for cross-border commerce, product research, supplier matching, and content generation.

## Ways to Contribute

You can contribute by:

- Improving documentation
- Suggesting new workflow ideas
- Reporting issues
- Creating product research templates
- Improving supplier matching logic
- Adding examples and use cases
- Helping design the future web interface

## Contribution Guidelines

Before contributing, please:

1. Open an issue to describe the idea or problem.
2. Keep contributions focused and easy to review.
3. Write clear documentation when adding new workflows.
4. Use simple and practical examples.
5. Be respectful and constructive in discussions.

## Project Direction

The goal of this project is not to build a complex enterprise platform immediately. The first goal is to create a clear, useful, and open-source foundation for AI-assisted commerce workflows.

## Maintainer

This project is maintained by an independent developer. Feedback, ideas, and practical suggestions are welcome.
